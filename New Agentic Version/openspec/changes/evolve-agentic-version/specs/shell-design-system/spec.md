## ADDED Requirements

### Requirement: Global application shell design
The system SHALL provide a dedicated view to design and preview the global application shell, including navigation and layout.

#### Scenario: View shell design
- **WHEN** user navigates to the Shell Design phase
- **THEN** system displays the interactive shell preview

### Requirement: Shell navigation configuration
The system SHALL allow the AI to define navigation items and user context that are reflected in the shell preview.

#### Scenario: Update shell nav
- **WHEN** the AI updates the shell specification with new navigation items
- **THEN** the shell preview reflects the updated navigation structure immediately
